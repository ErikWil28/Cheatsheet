Semantic HTML and You

Basic Website Notation

Tag

Name

Description

<header>

Header

Represents a container for introductory content or a set of navigational links.

<footer>

Footer

Defines a footer for a document or section.

<nav>

Navigation Link

Defines a set of navigation links

<main>

Footer

Specifies the main content of a document.

Semantics

Tag

Name

Description

<section>

Section

Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.

<article>

Article

Holds content that makes sense on its own such as articles, blogs, comments, etc.

<aside>

Aside

Contains information that is related to the main content, but not required in order to understand the dominant information.

<figure>

Figure

Encapsulates all types of media.

<figcaption>

Figure Caption

Used to describe the media in <figure>.

<video>

Video

Elements are used for media files.

<embed>

Embed

Elements are used for media files.

<audio>

Audio

Elements are used for media files.
